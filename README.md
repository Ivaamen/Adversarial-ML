# Adversarial-ML


This project demonstrates an existing method highlighted in the paper https://openaccess.thecvf.com/content_CVPRW_2019/papers/CV-COPS/Thys_Fooling_Automated_Surveillance_Cameras_Adversarial_Patches_to_Attack_Person_Detection_CVPRW_2019_paper.pdf, which shows how imperceptible changes to a particular image (either physically or digitally) can trick computer vision models into misclassifying information.
